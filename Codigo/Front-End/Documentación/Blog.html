Blog

Este componente es la vista del Blog, donde se crean los posts. Acá se utiliza el provider PostProvider que contiene las funcionalidades de crear, eliminar, cargar, etc. También se llaman a distintos componentes como son el sidebarBlog, el FormPost y el PanelPost.
